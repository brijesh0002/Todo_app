# 📝 To-Do App (React)

This is a simple To-Do app I built using React to practice building interactive UIs and managing state. The goal was to keep it clean, fast, and actually useful for everyday task tracking.

It’s not overly complicated—just something you can open and immediately start using to stay organized.

---

## 🚀 What it can do

- Add tasks quickly  
- Mark tasks as completed  
- Edit tasks if you make a mistake  
- Delete tasks you no longer need  
- Filter between All / Completed / Pending  

---

## 🛠️ Built with

- React (functional components + hooks)  
- JavaScript (ES6+)  
- CSS for styling  
- MongoDB (for storing and managing tasks)
